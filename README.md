# Retail Sales Data Analysis using SQL

## Project Overview
This project analyzes retail sales data using SQL to uncover customer purchasing behavior, product performance, and overall business revenue trends.  

The objective is to transform raw transactional data into meaningful business insights that can support strategic decision-making.

---

## Business Problem
Retail businesses need to understand:
- Who their most valuable customers are
- Which product categories generate maximum revenue
- How sales change over time
- Opportunities to improve profitability

This project answers these questions using structured SQL analysis.

---

## Dataset Description
The project uses a retail transaction dataset containing:

### Customers Table
- `customer_id`
- `age`
- `gender`
- `transaction_date`
- `transaction_id`

### Sales Table
- `transaction_id`
- `product_category`
- `quantity`
- `price_per_unit`
- `total_amount`

## Analysis Workflow

### Data Exploration
- Dataset overview
- Transaction value distribution
- Customer demographic analysis
- Data validation checks

### Customer Analysis
- High-value customers identification
- Age-group spending behavior
- Gender-wise spending analysis

### Product Analysis
- Revenue generated per product category
- Quantity sold by category
- Best-performing products
- High-volume but low-revenue categories

### Sales Performance Analysis
